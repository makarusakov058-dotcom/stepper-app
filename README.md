# Stepper — Шагомер

## 🚀 Сборка APK через GitHub Actions

### Шаг 1: Загрузи файлы на GitHub
1. Создай репозиторий `stepper-app`
2. Загрузи ВСЕ файлы из этой папки (по отдельности, не ZIP!)
3. Жми "Commit changes"

### Шаг 2: Запусти сборку
1. Перейди во вкладку "Actions"
2. Жми "Build APK" → "Run workflow" → "Run workflow"
3. Жди 10-15 минут

### Шаг 3: Скачай APK
1. Во вкладке "Actions" кликни на зелёную галочку
2. Внизу "Artifacts" → "stepper-apk" → "Download"

## 📱 Установка
Перекинь APK на телефон и установи!
